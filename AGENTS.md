# AGENTS.md

## Project

Cosmic Web is a Next.js site using TypeScript, Tailwind CSS, App Router, and pnpm.

## Working Guidelines

- Keep the project simple and maintainable.
- Do not add a database, CMS, authentication, or admin panel unless explicitly requested.
- Prefer local content in `src/content` for the blog until the content model requires a CMS.
- Avoid new dependencies unless there is a clear reason.
- Do not commit, push, merge, deploy, or open pull requests unless explicitly requested.

## Validation

- Run `pnpm lint` after code changes.
- Run `pnpm build` before handing off larger changes.
