# Cosmic Studio Project Brief

Cosmic Studio is a digital solutions studio built with Next.js, TypeScript, Tailwind CSS, and the App Router.

The website explains how Cosmic Studio designs and develops digital solutions for brands, businesses, and projects that need to operate better.

Cosmic Studio is the primary brand of the website.

## Current Scope

- Public website with a first visual direction for Cosmic Studio.
- Homepage focused on leads, clarity, digital operations, services, method, NODUX as a case/product, blog, and contact.
- Blog structure based on local project content.
- Clean component folders for layout, sections, and reusable UI.
- No database, CMS, authentication, or admin panel at this stage.
- NODUX appears only as a product, case, or laboratory built by Cosmic Studio.
- AI appears as an applied capability for digital systems, automation, and operations.

## Visual Direction

- Dark control-room and orbital-system aesthetic.
- Subtle cyan accents, modular panels, dashboard-like cards, grids, and system-status microcopy.
- Clear, strategic, human tone without looking like a generic marketing agency or cold software provider.
- AI language must stay operational and concrete, not generic or hype-driven.

## Future Options

- Move blog content to MDX if richer authoring is needed.
- Migrate content to Supabase or a CMS if editorial workflows grow.
- Add deployment documentation when the Vercel and GitHub flow is defined.
