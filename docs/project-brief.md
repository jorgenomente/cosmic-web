# Cosmic Web Project Brief

Cosmic Web is an initial website foundation built with Next.js, TypeScript, Tailwind CSS, and the App Router.

## Current Scope

- Public website with a simple homepage.
- Blog structure based on local project content.
- Clean component folders for layout, sections, and reusable UI.
- No database, CMS, authentication, or admin panel at this stage.

## Future Options

- Move blog content to MDX if richer authoring is needed.
- Migrate content to Supabase or a CMS if editorial workflows grow.
- Add deployment documentation when the Vercel and GitHub flow is defined.
