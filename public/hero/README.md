# Hero assets

Place optional local hero media here:

- `cosmic-hero-bg.webp`
- `cosmic-hero-bg.jpg`
- `cosmic-hero-bg.mp4`

The current hero uses a CSS fallback. To enable a static local image, set the `--cosmic-hero-image` custom property in `src/app/globals.css`.

Example:

```css
.cosmic-space-scene {
  --cosmic-hero-image: url("/hero/cosmic-hero-bg.webp");
}
```
